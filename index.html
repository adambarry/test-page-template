Button toolkit (demonstration)

To give a sensible example of what to do with a test-page like this this test-section contains an example of a button toolkit, in which all of the various button types are displayed in their possible states. This way, you always have a way of viewing all of the possible states for each button type without having to interact with the buttons first.

data-state attributes added to simulate states

In the following test-cases data-state attributes have been added to the appropriate elements in order to permanently display various states for the .btn element, e.g. hover, focus and active.
